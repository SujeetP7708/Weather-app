🌦️ Weather App

A responsive Weather Forecast Web Application built using HTML, Tailwind CSS, and JavaScript.
It fetches real-time weather data and 5-day forecasts using the OpenWeather API.

🚀 Features
🔍 Search weather by city name
📍 Get weather using current location (Geolocation API)
🌡️ Toggle between Celsius (°C) and Fahrenheit (°F)
📅 5-Day Weather Forecast
💨 Displays:
Temperature
Feels like
Humidity
Wind speed
Pressure
⚠️ Extreme heat alert (above 40°C)
🕘 Recent searches (stored using LocalStorage)
⌨️ Press Enter to search
📱 Fully responsive UI (Tailwind CSS)
🛠️ Tech Stack
Frontend: HTML, Tailwind CSS
JavaScript: Vanilla JS (ES6)
API: OpenWeather API
Storage: LocalStorage

⚙️ Setup & Installation
Clone the repository:
git clone https://github.com/SujeetP7708/Weather-app
Navigate to the project folder:
cd weather-app
Open index.html in your browser
🔑 API Configuration

This project uses OpenWeather API.

Go to: https://openweathermap.org/api
Generate your API key
Replace in script.js:
const apiKey = "YOUR_API_KEY";
